When we run JavaScript code, our code is received by the parser. The parser reads the codes line by line looking for errors & makes an Abstract Syntax Tree. Here variables & functions are hoisted. JavaScript also checks the variable type here. Then the code converts it into bytecode or machine code. This byte code is finally converted into output

JavaScripts executes code in single thread. Every statement is pushed to the call stack for execution. After finishing the execution of one statement it is removed from the call stack & another statements executes. This process goes on. When an event occurs it goes to the event queue and waits. When the call stack is empty event loop sends the code from the event queue to the call stack to executes the code.

The only difference between local storage & session storage is local storage holds the data until data is not removed. But data stored in session storage is removed automatically when the window is closed or refreshed.
